# Data Sources for NIS2 Threat Monitor

## Primary Sources
1. **MSB – Myndigheten för samhällsskydd och beredskap**
   - Nyheter / RSS-flöden
   - Incidentrapporter (där publikt)
   - Krisinformation

2. **CERT-SE**
   - Säkerhetsvarningar
   - Sårbarhetsnotiser
   - Incidentrapporter

3. **NCSC (Säkerhetspolisen + Försvarsmakten + MSB)**
   - Hotbedömningar
   - Teknik- och taktikrapporter

4. **ENISA**
   - Threat Landscape Reports
   - Deep dive-analyser
   - EU NIS2 guidance

## Secondary Sources
- CISA Alerts (USA, bra komplement)
- CVE/NVD Databases (sårbarheter)
- GitHub Security Advisories

## Metadata att extrahera
- Titel
- Källa
- Publiceringsdatum
- Kategori
- Hottyp
- Länk
- Sammanfattning
